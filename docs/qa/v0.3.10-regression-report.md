# LumenMark v0.3.10 回归测试报告

## 范围

- 全应用主题导入与切换。
- 设置菜单归类、主题预览、应用和删除。
- 深色代码块、LaTeX 公式块和工具按钮白底修复。

## 主题兼容限制

- v0.3.10 仍只支持导入单个 `.css` 文件。
- 出于隐私和稳定性考虑，导入 CSS 中的 `@import`、远程 URL、本地绝对路径和相对资源 `url(...)` 会被移除。
- 完整 Typora 主题文件夹导入、字体和图片 assets 复制与路径重写放入中远期版本。
