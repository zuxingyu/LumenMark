## 已知风险

- v0.3.9 只支持导入单个 `.css` 文件，不支持完整 Typora 主题文件夹；完整文件夹导入已列入中远期规划。
- 为保护隐私和稳定性，导入 CSS 中的网络资源、本地绝对路径、相对资源 URL 均会被禁用，因此依赖字体或图片资源的 Typora 主题可能只呈现基础排版。
- PDF 长文分页仍基于 canvas 截图分页，复杂长文可能存在分页位置不够理想的问题。
- 前端 bundle 仍包含 Mermaid/Milkdown 等较大依赖，Vite 可能提示 chunk 体积警告；该警告不代表功能失败。
- macOS 本地 app 包为 ad-hoc 签名，未 Apple 公证；Windows 安装包仍为未签名测试包。

## 结论

v0.3.9 计划内自动化验证、构建、Tauri 命令测试和敏感信息扫描通过。发布前仍需合并到 `main`、创建 `v0.3.9` 标签、推送并确认 GitHub Release 产物。
