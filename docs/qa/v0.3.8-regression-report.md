## 已知风险

- PDF 长文分页仍基于 canvas 截图分页，复杂长文可能存在分页位置不够理想的问题；本版本优先修复“无法导出/空白/报错”的稳定性问题。
- 前端 bundle 仍包含 Mermaid/Milkdown 等较大依赖，Vite 可能提示 chunk 体积警告；该警告不代表功能失败。
- macOS 本地 app 包为 ad-hoc 签名，未 Apple 公证；Windows 安装包仍为未签名测试包。

## 结论

v0.3.8 计划内自动化验证、构建、桌面启动冒烟和敏感信息扫描通过。发布前仍需合并到 `main`、创建 `v0.3.8` 标签、推送并确认 GitHub Release 产物。
