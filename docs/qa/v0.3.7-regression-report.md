## 已知风险

- PDF 长文分页仍基于 canvas 截图分页，复杂长文可能存在分页位置不够理想的问题；本版本优先修复“无法导出/空白/导出代码块”的稳定性问题。
- 前端 bundle 仍有较大的 Mermaid/Milkdown chunk，Vite 构建会提示体积警告；不影响本次功能正确性。
- macOS 本地 app 包为 ad-hoc 签名，未 Apple 公证；Windows 安装包仍为未签名测试包。

## 结论

v0.3.7 计划内自动化验证通过。发布前仍需完成敏感信息扫描、合并到 `main`、创建 `v0.3.7` 标签、推送并确认 GitHub Release 产物。
